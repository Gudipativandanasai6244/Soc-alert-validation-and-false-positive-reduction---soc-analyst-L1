# SOC Alert Validation and False Positive Reduction

## Project Objective

To monitor security events in Splunk, investigate alerts, validate false positives, and document the SOC Analyst L1 investigation process.

## Tools Used

* Splunk Enterprise
* Windows Event Logs
* GitHub

## Investigation Process

1. Monitored security events.
2. Searched logs using Splunk queries.
3. Reviewed timestamps and sources.
4. Investigated event details.
5. Determined whether alerts were true positives or false positives.

## Sample Output

Search Query: index=*

Result: 428 events found.

## Investigation Result

The investigated event showed no malicious indicators and was classified as a false positive.
